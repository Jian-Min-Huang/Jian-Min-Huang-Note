### install npm & gulp
```sh
$ sudo npm install -g gulp

$ npm install gulp
$ npm install gulp-file-include
$ npm install gulp-less
$ npm install gulp-concat
$ npm install gulp-uglify
$ npm install gulp-rename
```

### run gulp after you update content
```sh
$ gulp
```

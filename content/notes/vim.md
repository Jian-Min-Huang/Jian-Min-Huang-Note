##### Q : basic operation
```
i 
u
:q
:w
:e
:q!
```

##### Q : search text forward and backward
```
/keyward and ?keyword
```

##### Q : move to next keyword and last keyword
```
n and N
```

##### Q : jump to document head and bottom
```
gg and G
```

##### Q : copy, cut and paste
```
nyy, ndd and p
```

##### Q : replace
```
:s/old/new
:10,20s/old/new/g
:%s/old/new/g
```
